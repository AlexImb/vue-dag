# Vue DAG Builder

Data-driven directed acyclic graph (DAG) visual builder for Vue.js


## Installation

```bash
npm install --save vue-dag
```
or 
```bash
yarn add vue-dag
```

Import and register the component
```js
import VueDag from 'vue-dag';
```

```js
components: {
    VueDag
}
```

Use it in templates
```html
<vue-dag></vue-dag>
```
